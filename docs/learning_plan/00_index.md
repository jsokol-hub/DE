# Learning Plan Index

This folder contains the study plan for the first stage of the DE learning project.

## Files

- `01_study_rhythm.md` — weekly study structure tied to calendar blocks
- `02_resource_stack.md` — core learning resources and how to use them
- `03_roadmap_16_weeks.md` — high-level 16-week roadmap
- `04_start_todo.md` — first practical tasks and Cursor prompts
- `week_01.md` — detailed plan for week 1
- `week_02.md` — detailed plan for week 2
- `week_03.md` — detailed plan for week 3
- `week_04.md` — detailed plan for week 4

## How to work

1. Start each week by opening the relevant week file.
2. Use Tuesday for the main project task.
3. Use Wednesday for SQL and modeling study.
4. Use Thursday for Python ETL implementation.
5. Use Sunday to review and plan.
